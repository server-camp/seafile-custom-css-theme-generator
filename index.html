Seafile Custom CSS Theme Generator

Transform your Seafile service with a personalized color theme using custom CSS!

With this color configurator, you can quickly generate customized CSS code tailored to your preferred color scheme. Start by selecting a primary color, a primary text color, your Seafile version, and your preferred theme. Once you make your selections, you'll receive your custom CSS code. To apply it, go to the Seafile administration interface and enter the CSS code under “Settings” ➞ “Branding” ➞ “Custom CSS”.
